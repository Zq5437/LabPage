# 部署方式
本项目提供了不同的项目部署方式，针对不同实验室的需求，选择其中一项部署即可，点击跳转至相应文档。
只有第一种方式会无法使用`在线留言`功能和`访问统计`功能。
1. [使用**GitHub Pages**部署，<font color=red>零成本！</font>](./wiki/deploy_wiki/DEPLOYMENT_GITHUB_PAGES.md)
2. [使用**PC+内网穿透**部署，需有内网穿透账号，如cpolar等。](./wiki/deploy_wiki/DEPLOYMENT_NAT.md)
3. [使用**服务器**部署，需有专门的服务器。](./wiki/deploy_wiki/DEPLOYMENT_SERVER.md)
